# 🚗 TSIS 3 — Racer Game (Tkinter Version)
## 📌 Project Description
This project is a simple 2D racing game built using Python Tkinter without external game libraries like Pygame.  
The game includes dynamic road animation, obstacles, coins, scoring system, and persistent leaderboard using JSON.
---
## 🎮 Features
- 🚗 Player car controlled with arrow keys  
- 🛣 Animated road movement  
- 💰 Coin collection system (+10 score per coin)  
- 🚧 Random obstacles (game over on collision)  
- 📊 Score + distance tracking  
- 💾 Persistent leaderboard using JSON  
- 🏁 Game Over system  
- 🎯 Lane-based movement system  
---
## 🧠 Technologies Used
- Python 3  
- Tkinter (built-in GUI library)  
- JSON (data storage)  
- Random module
